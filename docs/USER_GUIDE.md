# User Guide

## Roles
- HR Accrual Admin: Configure settings, components, run accruals
- HR Accrual Operator: Maintain employee accrual rows, run accruals
- Finance Accrual Controller: Review postings, audit trail
- Accrual Auditor: Read-only

## Monthly processing
1. Open HR Accrual Management workspace.
2. Create an Accrual Run (Company, Posting Date).
3. Click "Process Accruals".
4. System creates Journal Entries and stores links in the run log.

## Audit
Use Accrual Run logs to trace employee-level accrual amounts to Journal Entries.
